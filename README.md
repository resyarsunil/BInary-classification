# BInary-classification
Dataset:  Breast cancer dataset   
269 rows and 31 columns 
Breast cancer is the target variable: 1 for Malignant and 0 for Benign 
All the variables are numerical variable  

Here, Random Forest is used for the binary classification 

70% of the dataset is used for training and 30% for testing 

For evaluation, metrics such as accuracy and confusion metrics are considered
